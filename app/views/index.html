About this prototype

This prototype has 2 main sections: product pages and admin tool.

Product page sections

Product homepage

This page gives a broad overview of the platform, what it can do and why a service should use it.

Get started

Outlines the steps the user needs to take to start using GOV.UK PaaS.

Request an account

This is where the user requests a PaaS account for their organisation.

Features section

The features section has multiple sub pages such as: Roadmap, Information assurance and Security

Pricing section

This section gives an overview of the pricing model and links to the pricing calculator.

Pricing calculator

The user can select different backing services and work out how much their service is likely to pay for using GOV.UK PaaS.

Support

Outlines the different support options for the user.

Admin tool sections

My organisations

List the organisations the user has admin access too.

Dashboard

Gives an overview of a specific organisations PaaS usage and allows the user to dig deeper.

Team members

Gives an overview of a specific organisations team members and permissions. Gives the user the opportunity to add more team members or view a specific team member in more detail.

Billing section

Allows the user to see what their service has spent on using GOV.UK PaaS on a monthly basis.
